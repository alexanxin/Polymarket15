To create an OG image, you can:

1. Use Canva.com or Figma to create a 1200x630px image
2. Include elements like:
   - Dark background (matching the app theme)
   - 'Polymarket Trading Bot' title
   - Icons for crypto currencies (BTC, ETH, SOL, XRP)
   - Trading charts or market data visualization
   - Professional color scheme (blues, greens, dark theme)

3. Save it as 'og-image.png' in the 'public' folder

4. Replace 'https://your-domain.com' in layout.tsx with your actual domain
5. Replace '@your_twitter_handle' with your Twitter handle

Example tools:
- Canva: Search for 'social media post' template
- Figma: Create from scratch
- Remove.bg: For transparent backgrounds
- Unsplash/Pexels: For background images
